# Demo GIF

The README's demo GIF is generated from `demo.tape` with
[VHS](https://github.com/charmbracelet/vhs) so it's reproducible.

```bash
# 1. install VHS (one-time): https://github.com/charmbracelet/vhs#installation
# 2. from the repo root:
vhs demo/demo.tape          # writes docs/demo.gif
# 3. commit docs/demo.gif, then uncomment the image line near the top of README.md
```

No VHS? Any screen recorder works — just run the two commands in `demo.tape` and
save the result as `docs/demo.gif`.
